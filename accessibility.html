Accessibility Statement

Last updated: February 2026

Accessibility isn't an afterthought here — it's the whole point. I'm an accessibility consultant who has worked directly with students with disabilities for seven years. This site is built to the same standards I hold my clients to.

WCAG 2.1 Level AA Full conformance

What I've done

This website has been built and tested to conform with the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. Specific measures include:

  • Semantic HTML throughout — proper heading hierarchy, landmark regions, form labels, and link text
  • Full keyboard navigability with visible focus indicators on every interactive element
  • Color contrast ratios meeting or exceeding WCAG AA thresholds (minimum 4.5:1 for text, 3:1 for large text and UI components)
  • Content reflows at 320 CSS pixels (400% zoom on a 1280px viewport) with no horizontal scrolling, per SC 1.4.10
  • Responsive design tested on mobile, tablet, and desktop screen sizes
  • No content relies solely on color to convey meaning
  • Animations respect the prefers-reduced-motion user setting
  • ARIA attributes used correctly per WAI-ARIA 1.2 — supplementing native semantics, never overriding them

Assistive technology

This site has been built and tested for compatibility with:

  • Keyboard-only navigation (Tab, Shift+Tab, Enter, Escape)
  • Browser zoom up to 400%
  • Screen readers (semantic HTML, ARIA landmarks, and proper labeling throughout)

It has also been manually reviewed for issues that automated testing commonly misses, including heading hierarchy, focus order, and meaningful link text.

Known limitations

I actively maintain this site and work to resolve issues as they arise. Currently, I'm not aware of any accessibility barriers. If you encounter one, please let me know — I take that seriously.

Feedback

If you experience any difficulty using this site, or if something doesn't work with your assistive technology, I want to hear about it. Accessibility isn't a checkbox — it's an ongoing commitment.
